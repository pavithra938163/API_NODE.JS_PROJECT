# School Management API

A backend project built using Node.js, Express.js, and MySQL.

## Features
- Add school API
- List schools by proximity
- JWT authentication
- Input validation with Zod
- Pagination support
- Search support
- Soft delete support
- Rate limiting
- Clean project structure
- Error handling
- Basic unit test placeholders

## Tech Stack
- Node.js
- Express.js
- MySQL
- JWT
- Zod
- Jest
